## Dev Notes

### OpenAI Client

[Source: architecture.md#Section 16.4]

```typescript
// src/lib/openai.ts
import OpenAI from 'openai';
export const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});
```

- Instalar: `npm install openai`
- API Key e server-only (NUNCA expor no client-side)
- Usar apenas em API Routes e Server Components

### Hierarquia de Componentes da Home

[Source: architecture.md#Section 8.2]


### Header e Navegacao

| Rota | Descricao |

- TypeScript strict mode obrigatorio
