## Story

**As a** desenvolvedor,
**I want** modelos Prisma para rotinas, passos, execucoes, dependencias e pendencias,
**so that** o banco de dados suporte todo o sistema de gestao de rotinas.

## Acceptance Criteria

1. 5 novos modelos no schema: Routine, RoutineStep, RoutineExecution, RoutineDependency, Pendencia
2. Migration executada com sucesso no banco PostgreSQL
3. Indices criados para campos de busca frequente (active, routineId, executedAt, status, priority)
4. Constraint unique em RoutineDependency (dependentId, dependencyId)
5. Cascade delete configurado (deletar rotina remove steps, execucoes e dependencias)
6. Types TypeScript adicionados: RoutineFrequency, RoutineStatus, RoutineWithStatus, PendenciaItem, MeuDiaCounters, etc.
7. `npm run typecheck` passa sem erros
