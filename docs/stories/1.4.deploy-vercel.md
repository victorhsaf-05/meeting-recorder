### Variaveis de Ambiente

[Source: architecture.md#Section 13.3]

| Variavel | Descricao |
|----------|-----------|
| `DATABASE_URL` | Connection string do Neon PostgreSQL |
| `OPENAI_API_KEY` | Chave de API da OpenAI |

- Configurar via Vercel Dashboard > Project > Settings > Environment Variables
- Marcar para todos os environments: Production, Preview, Development

### Build Command

[Source: architecture.md#Section 13.4]

```
prisma generate && next build
```

- O `prisma generate` e necessario para gerar o client antes do build
- Ja configurado no `package.json` na Story 1.2

### Restricoes do Vercel Free Tier

[Source: architecture.md#Section 3.2]

| Restricao | Limite |
|-----------|--------|
| Request body | 4.5 MB |
| Function timeout | 10 segundos |
| Bandwidth | 100 GB/mes |
| Builds | 6000 min/mes |

### Estrategia de Deploy

[Source: architecture.md#Section 13.1]

- **Production**: deploy automatico no push para `main`
- **Preview**: deploy automatico em pull requests
- **Rollback**: disponivel via dashboard da Vercel
- Git integration e o metodo recomendado para CI/CD
