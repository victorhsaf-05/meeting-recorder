## Story

**As a** usuario,
**I want** APIs para gerenciar pendencias e um endpoint de dashboard diario,
**so that** eu possa rastrear tarefas avulsas e ter visao consolidada do meu dia.

## Acceptance Criteria

1. GET `/api/pendencias` retorna pendencias com filtros por status e prioridade, mais counters por status
2. POST `/api/pendencias` cria pendencia com validacao de titulo
3. PUT `/api/pendencias/[id]` atualiza campos parcialmente
4. DELETE `/api/pendencias/[id]` remove pendencia
5. GET `/api/meu-dia` retorna: rotinas com status != OK, pendencias ativas (Pendente/Em andamento), e counters (totalRotinas, concluidasHoje, atrasadas, desatualizadas, pendenciasPendentes, pendenciasAndamento)
6. Counters de pendencias usam groupBy para eficiencia
