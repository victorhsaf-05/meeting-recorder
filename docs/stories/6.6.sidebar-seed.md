## Story

**As a** usuario,
**I want** navegacao atualizada com as novas paginas e dados iniciais para teste,
**so that** eu possa acessar as rotinas pela sidebar e testar o sistema imediatamente.

## Acceptance Criteria

1. Sidebar agrupada em secoes: "Menu" (Reunioes, Dashboard, Nova Gravacao) e "Rotinas" (Meu Dia, Rotinas, Pendencias)
2. Icones corretos: CalendarCheck (Meu Dia), ListChecks (Rotinas), ClipboardList (Pendencias)
3. Deteccao de ativo mantida com pathname === item.href
4. Seed cria 3 rotinas: Atualizar Razao (daily), Atualizar Banco de Dados (daily), Atualizar DRE (daily, depende de Razao)
5. Seed inclui steps para cada rotina
6. Seed inclui execucoes historicas (todas vencidas desde marco)
7. Seed inclui dependencia: DRE depende de Razao
8. DRE tem observacao com instrucoes de copia
9. `npm run build` passa sem erros
10. `npm run lint` passa sem erros novos
