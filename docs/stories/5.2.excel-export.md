### Funcao de Exportacao Completa

[Source: architecture.md#Section 20.1]

```typescript
// lib/excel.ts
import * as XLSX from 'xlsx';

export function exportTodosToExcel(todos: Todo[], filename: string) {
  const data = todos.map((t) => ({
    'Responsavel': t.responsible || '',
    'Responsavel pela acao': t.actionOwner || '',
    'Centro de custo': t.costCenter || '',
    'Data reuniao': formatDate(t.meetingDate),
    'Conta': t.account || '',
    'TO-DO': t.action,
    'Prazo': t.deadline ? formatDate(t.deadline) : '',
    'Status': t.status,
  }));

  const ws = XLSX.utils.json_to_sheet(data);
  const wb = XLSX.utils.book_new();
  XLSX.utils.book_append_sheet(wb, ws, 'To-Dos');
  XLSX.writeFile(wb, `${filename}.xlsx`);
}
```

Adicionar em `src/lib/utils.ts` se nao existir:

```typescript
export function formatDate(date: Date | string): string {
  const d = typeof date === 'string' ? new Date(date) : date;
  return d.toLocaleDateString('pt-BR');
}
```

### Client-Side Only

[Source: architecture.md#Section 2.1]

SheetJS roda inteiramente no browser â€” zero carga no servidor, sem limite de timeout. O componente `ExcelExport` deve ser um Client Component (`'use client'`).

### Nomes dos Arquivos

- **Dashboard:** `todos-{data}.xlsx` (ex: `todos-15-03-2026.xlsx`)

O titulo deve ser sanitizado (remover caracteres especiais) para uso no nome do arquivo.
